# CSS

## 概要
- Cascading Style Sheets の略
- スタイルシート言語に分類される
  - プログラミング言語ではない
- Web ページのスタイルを記述する
  - 文字を赤くする、大きさを調整する
  - サイドバーの大きさを調整する、スクロール可能か設定する
  - etc...
